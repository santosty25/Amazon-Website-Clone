Amazon Clone – E-Commerce Web Application
Overview

This project is a full-stack Amazon-style e-commerce web application that allows users to browse products, add items to a shopping cart, and complete a checkout process. 
The goal of this project was to learn how to build scalable front-end applications with React while integrating authentication, state management, and a backend service.

The application replicates key features of a modern e-commerce platform and demonstrates how front-end interfaces interact with backend services.

website link: https://clone-63f58.web.app/


Features
* User authentication
* Product browsing
* Add/remove items from cart
* Shopping cart state management
* Checkout page
* Responsive UI design
* Persistent user sessions


Tech Stack

Frontend
* React
* JavaScript / TypeScript
* CSS

Backend / Services
* Firebase Authentication
* Firebase Firestore / Database
